#  goexplore_sales_analysis
## Overview
This project was completed as part of a group data analysis challenge. GoExplore, a growing camping and hiking supplier, lost their only data scientist unexpectedly and needed urgent business insights. We were handed a single spreadsheet and tasked with delivering a full analysis using BigQuery and Looker Studio.
Business Questions

Do specialty stores (golf shops, eyewear stores) perform differently from general sports or outdoor shops in terms of average order value, product pricing, and order volume?
What market sizes should be expected in four new target countries — Czech Republic, Norway, Poland and Portugal — based on neighbouring markets GoExplore already operates in?
